# TheNewtownClasses

A full-stack web application built for **The Newtown Classes**, a coaching centre based in Newtown, Kolkata.
The app helps students easily browse available courses, check class schedules, and enroll online.

## Features
- Course listings with details
- Class schedule view
- Student enrollment
- Fully responsive design for mobile and desktop
- SEO integrated for better online visibility

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, React.js
- **Database:** MySQL / SQL
- **Other:** SEO Integration, REST APIs, Custom Domain Integration

## Getting Started
```bash
git clone https://github.com/WierdCodeWallah/TheNewtownClasses.git
cd TheNewtownClasses
npm install
npm start
```

## About
This project was built as a real-world web application to help a local coaching 
centre manage their online presence and make it easier for students to find and 
join classes.
